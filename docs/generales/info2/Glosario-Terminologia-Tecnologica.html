Conceptos Fundamentales

Toda gran estructura se construye sobre cimientos sólidos. Estos son los términos y metodologías transversales que forman la base del desarrollo de software moderno.

Metodologías de Desarrollo

Las metodologías organizan el caos del desarrollo en procesos estructurados. La agilidad permite adaptarse al cambio, mientras que el modelo en cascada requiere una planificación inicial exhaustiva.

Agile

Enfoque iterativo y flexible centrado en la colaboración y la entrega continua de valor. Saber más

Waterfall (Cascada)

Desarrollo Web

El ecosistema web se divide en dos mundos complementarios: lo que el usuario ve y con lo que interactúa (Front-End) y la lógica que lo hace funcionar por detrás (Back-End).

Los Roles en el Desarrollo Web

Entender la especialización es clave. El Front-End construye la fachada visible, el Back-End edifica la maquinaria interna, y el Full-Stack domina ambos terrenos, conectando la experiencia de usuario con la lógica del servidor.

Cloud Computing

La "nube" no es más que ordenadores de terceros accesibles por internet. Ofrece servicios que abstraen la complejidad de la infraestructura física, permitiendo a los desarrolladores centrarse en crear.

Modelos de Servicio "As a Service"

Cada nivel de la pirámide ofrece una capa mayor de abstracción. IaaS proporciona la infraestructura básica, PaaS añade la plataforma de desarrollo, y SaaS entrega el software final listo para usar.

DevOps y MLOps

Filosofías que buscan unificar el desarrollo (Dev), las operaciones (Ops) y el Machine Learning (ML) para automatizar y agilizar la entrega de software y modelos de IA.

El Ciclo Infinito de DevOps

DevOps no es una herramienta, sino una cultura de colaboración y automatización. Su ciclo de vida representa un proceso continuo de mejora, desde la planificación hasta el monitoreo y la retroalimentación.

Conceptos Clave de Operaciones

CI/CD (Continuous Integration / Continuous Delivery)

Práctica de integrar y entregar código de forma automática y frecuente para reducir errores y acelerar el despliegue. Saber más

IaC (Infrastructure as Code)

Gestionar y aprovisionar infraestructura mediante código y herramientas de automatización, en lugar de procesos manuales. Saber más

MLOps (Machine Learning Operations)

Aplicación de los principios DevOps a los flujos de trabajo de Machine Learning, para automatizar y gestionar el ciclo de vida de los modelos. Saber más

Cloud Native y Orquestación

Construir y ejecutar aplicaciones para aprovechar al máximo las ventajas del modelo de computación en la nube, centrándose en la resiliencia, la agilidad y la escalabilidad.

Kubernetes (K8s)

Sistema de orquestación de contenedores de código abierto para automatizar el despliegue, escalado y gestión de aplicaciones en contenedores. Saber más

Microservicios

Estilo arquitectónico que estructura una aplicación como una colección de servicios pequeños, autónomos y débilmente acoplados. Saber más

Service Mesh

Observability (Observabilidad)

Capacidad de entender el estado interno de un sistema a partir de sus salidas externas (Logs, Métricas, Trazas). No solo es monitorear, es preguntar. Saber más

Seguridad y Redes

Proteger los datos y la infraestructura es fundamental. Estos conceptos definen cómo se controla el acceso, se defiende contra amenazas y se garantiza la integridad de la información.

VPN (Virtual Private Network)

Crea una conexión de red privada y cifrada a través de una red pública, como Internet, para navegar de forma segura y anónima. Saber más

Firewall (Cortafuegos)

Sistema de seguridad que monitorea y controla el tráfico de red entrante y saliente basándose en reglas de seguridad predeterminadas. Saber más

Zero Trust (Confianza Cero)

Modelo de seguridad estratégico que asume que no se debe confiar en nada ni en nadie por defecto, ni siquiera dentro de la red corporativa. Saber más

Encryption (Cifrado)

Proceso de codificar información para que solo las partes autorizadas puedan acceder a ella, convirtiendo texto legible en texto ilegible. Saber más

Ciencia de Datos y Machine Learning

Disciplina que extrae conocimiento y valor de los datos. Utiliza técnicas de Machine Learning (una rama de la IA) para crear sistemas que aprenden y predicen.

Popularidad de Librerías de ML

El ecosistema de Machine Learning está impulsado por potentes librerías de código abierto. TensorFlow y PyTorch dominan en Deep Learning, mientras que Scikit-learn es el estándar para ML tradicional.

La Relación entre AI, ML y DL

La Inteligencia Artificial es el conjunto de software que automatiza la aplicación de técnicas avanzada de ML y DL. El Machine Learning es una subdisciplina de la computación y la Statistical Learning Theory (en azul claro) que permite a las máquinas aprender de los datos. El Deep Learning (DP) es una técnica avanzada de Machine Learning (ML, en azul cielo) basada en redes neuronales profundas (D-ANN).
